# Meine erste Website

Dies ist dein Startprojekt im Enrichment-Kurs *„Mit KI zur eigenen Website“*.

## Dateien
- **index.html** – Struktur der Seite  
- **style.css** – Farben, Layout und Schriftarten  
- **script.js** – Interaktivität mit JavaScript  

## Ziel
Passe den Text, die Farben und den Inhalt an dein eigenes Thema an.


Wenn du fertig bist, kannst du deine Seite über **GitHub Pages** veröffentlichen:
> Settings → Pages → Branch: `main` → `/ (root)` → Save.
Lade nun die Seite neu und du siehst: (Visit Site) → Der Knopf bringt dich zu deiner Website.
